## Common Questions (Prep Answers)
Q: "Does this work with private repos?"
A: Yes! Just provide a GitHub token with repo access. All processing is local.

Q: "What about other platforms (GitLab, etc)?"
A: GitHub first, but we're open to PRs for other platforms!

Q: "Can I customize the report format?"
A: Custom templates coming soon! For now, it's optimized for clarity and shareability.

Q: "Is my data private?"
A: 100%. Uses public data by default, local processing, no analytics or tracking.

Q: "How is this different from GitHub's contribution graph?"
A: GitHub shows *activity*. We show *impact*. Features vs fixes, meaningful categorization, team views.
