# Team Overview Template

Example command:
```bash
./whatdidyougetdone.py team user1 user2 user3 --days 7
```

## Purpose
Aggregate activity across multiple team members for team reports and management.

## Format
- Combined statistics for all team members
- Individual breakdowns per person
- Cross-repository visibility
- Team productivity metrics
