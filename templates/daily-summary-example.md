# Daily Summary Template

Example command:
```bash
./whatdidyougetdone.py report <username> --days 1
```

## Purpose
Quick overview of today's contributions.

## Format
- Focused on today's activity only
- Highlights key contributions
- Ideal for daily standups or end-of-day reviews
